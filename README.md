## IMPORTANT

__Use nodejs <= v0.10.x__

### Setup

#### nvm users

There's a `.nvmrc` file on the project root directory, so:

```
nvm install
nvm use
```

#### Install dependencies

```
npm install -g sails
npm install
```

### Run!

```
sails lift
```
